# with-async-plugin

`humming` app using `createApp()` with an async plugin.

## Run

```bash
bun run example:with-async-plugin
```

## What it shows

- `createApp()` for async plugin setup
- async `setup()` support in `definePlugin()`
- route registration after async initialization

## Endpoints

- `GET /health`
- `GET /api/ready`
