# humming Brand Assets

This folder contains the current brand kit for `humming`.

## Files

- `humming-logo.svg`: Horizontal primary logo for docs, headers, and release visuals.
- `humming-logo-readme.svg`: High-contrast horizontal logo variant tuned for GitHub README dark backgrounds.
- `humming-mark.svg`: Standalone mark for avatars, favicons, app icons, and badges.

## Brand Direction

- Positioning: plugin-first lightweight BFF kernel.
- Personality: fast, calm, dependable, and developer-friendly.
- Visual metaphor: a compact core with clear extension points.
- Shape language: bracket rails, swift wing geometry, rounded forms, restrained plugin cue.

The current mark is the only active brand direction. A hummingbird sits inside a pair of bracket-like rails, expressing a light BFF layer with a clear boundary and a fast request flow.

## Color Tokens

- `#10244A` midnight blue
- `#0A1531` deep navy
- `#54D9F2` electric cyan
- `#79D9FF` bracket blue
- `#FFB34A` amber
- `#FF8E6B` coral plugin accent

## Typography Suggestion

- Display: `Space Grotesk` or `Sora`
- Body: `IBM Plex Sans` or `Noto Sans`

## Usage Guidelines

- Prefer `humming-logo.svg` for README headers, docs homepages, and presentations.
- Use `humming-mark.svg` anywhere a compact product identifier is needed.
- Keep at least 12% clear space around the outer bounds of the mark.
- Preserve the relationship between the bracket rails, bird form, and plugin node.
- Avoid adding alternate mascot versions or parallel logo styles.

## Design Notes

- The left and right rails act like brackets, expressing the BFF boundary that receives and returns traffic.
- The bird carries speed, lightness, and routing agility.
- The soft outer stroke and shadow refine the panel edge so the mark feels cleaner on dark and transparent backgrounds.
- The composition stays readable at small sizes because the bracket rails remain visually stable even when the inner details shrink.

## Export Tips

- Export PNGs at `2x` and `3x` from the SVG sources for web use.
- Generate favicon/app icon sizes from `humming-mark.svg`: `32`, `64`, `128`, `256`, `512`.
- Treat the SVG files as the single editable source of truth.
